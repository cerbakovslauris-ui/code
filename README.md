3D Horror Maze Game

Apraksts

Šis projekts ir 3D pirmās personas šausmu spēle, izveidota ar WorldViz Vizard platformu.
Spēlētājs atrodas tumšā labirintā un viņa mērķis ir atrast trīs atslēgas, lai atvērtu durvis un nonāktu pie izejas.
Spēle piedāvā atmosfērisku vidi ar lukturīti, spriedzes skaņām, jumpscare mehāniku un staminas sistēmu.


Sistēmas prasības

Windows OS
